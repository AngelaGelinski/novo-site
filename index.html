Os ramos são ramificaçoes de um projeto de software ou documento, tendo com objetivo o desenvolvimento paralelo de:

  • novas de funcionamentos
  • Coreção de problemas
  • alteração de codigo

Podemos colocar o projeto na Branch Master ou Main que é o braço principal do projeto de repositório e no Branch Secundaria que é o braço secundario de um projeto em um repositorio .

Na sequencia o video mostra como fazer as raficaçoes no GitHub, primeiro de tudo, você precisa entrar no Current Barnch, ai escreve no filter, como secundario e clica no new branch, em seguida mostra como juntar as ramificações
